# 🚀 Most Common (Quick Reference)

## ✅ Success

- `200` → OK
- `201` → Created
- `204` → No Content

## ⚠️ Client Errors

- `400` → Bad Request
- `401` → Unauthorized
- `403` → Forbidden
- `404` → Not Found
- `409` → Conflict
- `422` → Validation Error
- `429` → Too Many Requests

## 💥 Server Errors

- `500` → Server Error
- `502` → Bad Gateway
- `503` → Service Unavailable
- `504` → Timeout
